# MGVF-U Interactive Explorer

A lightweight browser-based black hole cockpit simulator.

## Files

- `index.html` - main web page
- `style.css` - visual design and cockpit styling
- `script.js` - simulator controls and clock logic
- `assets/` - placeholder folder for future images, audio, or icons

## How to run locally

Open `index.html` in any modern browser.

## How to share as a working link

### Netlify, easiest
1. Go to Netlify.
2. Drag this entire folder onto the Netlify deploy area.
3. Netlify gives you a public web link.

### GitHub Pages
1. Create a new GitHub repository.
2. Upload these files.
3. Go to Settings > Pages.
4. Deploy from the main branch.
5. Open the GitHub Pages URL.

## Note

This is a conceptual educational simulator. General-relativistic time dilation is the established physics layer. The vacuum stiffness field is an MGVF-U conceptual visualization.
